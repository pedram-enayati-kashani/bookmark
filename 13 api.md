#### telegram

**getME :**
```
https://api.telegram.org/bot<token>/getMe
```


**getUpdates :**
```
https://api.telegram.org/bot<token>/getUpdates
```

**sendMessage :**
```
https://api.telegram.org/bot<token>/sendMessage?chat_id=254133636&text=Hello+world
```
